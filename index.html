Ini adalah kalkulator sederhana untuk menghitung luas dan keliling segitiga

Luas Segitiga

Rumus Luas Segitiga yaitu :

L = 1/2 x a x t

  • Di mana :
  • L = Luas
  • a = Panjang Alas
  • t = Tinggi

Hitung Luas Segitiga

Keliling Segitiga

Rumus Keliling Segitiga yaitu :

K = S1 + S2 + S3

  • Di mana :
  • K = Keliling
  • S1 = Sisi A
  • S2 = Sisi B
  • S3 = Sisi C
